# Agent.md - AI Usage Documentation

This project was built in collaboration with **Antigravity**, an AI pair-programming assistant designed by the Google DeepMind team.

## 🤖 How AI Was Used

1. **Architecture & Planning**: Used to structure the database schema, API routes, choose libraries like React Flow, and generate project boilerplate.
2. **Algorithm Implementation**: Helped implement the BFS based failure simulation and DFS based circular dependency checks.
3. **Frontend & Styling**: Assisted in building the interactive canvas, sidebar panels, and custom CSS layouts.
4. **Environment Setup**: Configured environment variable management, and project `.gitignore` patterns.
5. **Research & Modeling**: Utilized **Claude** for algorithm modeling, architectural options research, and verifying graph traversal corner cases.
